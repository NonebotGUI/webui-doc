# 安装依赖

::: warning
NoneBot WebUI 只是工具，所以并不提供 `nb-cli` 的安装，需要用户手动安装依赖
:::

## 环境要求

- Windows10 及以上
- GNU/Linux 任意发行版
- MacOS

::: danger
`macOS`未经过测试，可能无法使用
:::
## Python

目前（2024年）， NoneBot 已经取消对 `Python 3.8` 的支持，所以请确保你安装的 Python 版本 **>=3.9**

## NoneBot CLI

即 `nb-cli`， nbgui 提供 nb-cli 来进行对 bot 的操作。在安装好 Python 后使用以下命令安装：

```
pip install nb-cli
```
